FINANCIAL INFORMATION USER PROMPTS

Use one section block at a time.
Copy only the block for the target section.

GLOBAL RULES FOR ALL BLOCKS
- Output only the requested section.
- Preserve the heading exactly.
- Use the accountant report and retrieved evidence as the only factual source.
- Use the style reference only for structure and tone.
- Ignore figures, percentages, dates, line items, note numbers, and cross-references in the style reference.
- If the section is wholly unsupported, output exactly: Not disclosed in the provided documents.
- Treat the section as wholly unsupported only when the evidence contains no usable content for the target section after reviewing all supplied snippets.
- If the evidence supports only part of the requested section, draft only the supported part and omit the unsupported rows or statements.
- Do not use the fallback wording when the evidence supports a partial table, partial narrative, or only some rows or periods.
- Do not treat orphan line-item labels without aligned figures, period headers, or directly usable note content as enough support for a statement table.
